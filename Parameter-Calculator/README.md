# Parameter Calculator `v3.0.1`

Counts real and virtual DMX parameters of all fixtures in the show, including multi-instance geometry handling. Calculates how many grandMA3 Parameter Units (PU M/L/XL) are needed to cover the show.

**[← Back to all plugins](../README.md)**

---

## Features

- Counts **real** and **virtual** parameters for all patched and unpatched fixtures
- Handles multi-instance geometry correctly
- Calculates missing parameters and recommends the minimum PU combination needed
- Supports FullSize, Light, and OnPc station configurations
- Add PU M / L / XL units to calculate total available parameters
